WiFi Pineapple
Hak5 / Wireless Auditing Platform

A purpose-built wireless auditing platform that masquerades as a legitimate access point. It responds to probe requests from nearby devices, luring them to auto-connect. Once connected, it intercepts traffic, serves evil portals, captures credentials, and runs MITM attacks — all via a polished web dashboard. The Mark VII fits in a backpack and targets dozens of devices simultaneously with no technical expertise required.

Shark Jack
Hak5 / Ethernet Implant

A compact, battery-powered network attack tool disguised as an Ethernet adapter. Plug it into a live network port and it immediately executes payloads — scanning, fingerprinting, and exfiltrating network intelligence within seconds. Its companion, the LAN Turtle, installs as a transparent USB Ethernet adapter enabling persistent remote access via SSH or reverse tunnels, punching silently through firewalls.

O.MG Cable
Covert Hardware / HID Implant

Visually and functionally identical to a standard USB cable — down to the moulding and weight. Hidden inside is a full WiFi-capable implant and microcontroller. When plugged in, it injects keystrokes, runs scripts, exfiltrates data over WiFi, self-destructs its payload to defeat forensic analysis, and maintains a persistent backdoor. It bypasses OS-level USB security by presenting as a legitimate HID keyboard.

USB Rubber Ducky
Hak5 / Keystroke Injection

The device that pioneered keystroke injection attacks. It presents as a trusted USB keyboard, bypassing every security tool that blocks unknown USB storage. In seconds it types hundreds of commands — opening terminals, downloading malware, exfiltrating data, creating backdoor accounts, or disabling security software. Ducky 3.0 includes conditional logic so payloads adapt to the detected OS and environment automatically.

Nmap
Software / Network Scanner

The industry-standard open-source tool for network discovery and security auditing. Uses raw IP packets to determine what hosts exist, what services they run, what OS they use, and what firewalls are in place. While administrators use it daily for legitimate audits, attackers use it in reconnaissance — quietly mapping a target network before exploitation. Advanced scan types can evade basic intrusion detection entirely.

Flipper Zero
Multi-Tool / RF, NFC & HID

A portable multi-tool for hardware hacking packaged in a playful dolphin-themed device. Reads, clones, and replays RFID cards (including building access badges), NFC tags, IR remotes, and sub-GHz radio signals. GPIO pins allow direct hardware interfacing. BadUSB mode functions like a Rubber Ducky. Received major media attention for interacting with everything from garage doors to contactless payment terminals, sparking legislative discussions in multiple countries.
